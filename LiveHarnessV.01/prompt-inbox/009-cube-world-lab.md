# Product Requirements

Create a browser-playable cube world with:

- block terrain
- player movement
- place and remove actions
- selected block palette
- inventory state
- objective state
- window.GameHost.getState() for inspection

Use NexusRealtime domain terms for build, break, inventory, movement, and save snapshot concepts.
